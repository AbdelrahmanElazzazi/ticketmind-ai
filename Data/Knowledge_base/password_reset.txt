Password Reset Procedure

Users who forget their passwords can reset them through the self-service password recovery workflow available on the login page. The process begins when the user clicks the "Forgot Password" link and enters the email address associated with the account.

Once the request is submitted, the system generates a secure password reset token and sends an email containing a reset link. For security purposes, the link remains valid for 30 minutes and can only be used once.

Users should verify that the email address entered matches the registered account. If no email is received within five minutes, users should check spam, junk, and quarantine folders before contacting support.

Password requirements include a minimum length of 12 characters, at least one uppercase letter, one lowercase letter, one number, and one special character. Reusing the previous five passwords is not allowed.

If the reset link has expired, users must generate a new request. Multiple reset requests invalidate previously generated links.

Support agents cannot view user passwords. In exceptional situations where a user cannot access their registered email address, identity verification must be completed before updating account information.

Common Issues:

- Password reset email not received
- Reset link expired
- Invalid reset token
- Email address not registered
- Password complexity requirements not met

Troubleshooting Steps:

1. Confirm email address.
2. Check spam folder.
3. Generate a new reset request.
4. Verify password policy compliance.
5. Escalate unresolved issues to account administration.

Expected Resolution Time:
Most password reset requests are completed within 10 minutes.
